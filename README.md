# Pengantar
Repositori ini menyimpan [salindia](https://github.com/dumilah/dumilah-zenodo-tutorial-slides/blob/main/GPWRajeg_2026_DUMILAH-Tutorial-Zenodo.pdf) tutorial penggunaan Zenodo (European Organization For Nuclear Research & OpenAIRE, 2013) dalam rangka peluncuran *[DUMILAH (Digital Humanities Landscape at the Faculty of Humanities)](https://fib.unud.ac.id/pages/digital-humanities)*. Tutorial audiovisual untuk Zenodo juga tersedia di [laman utama *DUMILAH*](https://fib.unud.ac.id/pages/digital-humanities#dumilah-zenodo). Tim dari Zenodo juga menyediakan bahan pengenalan untuk Zenodo seperti salindia (Nielsen, 2017) dan poster (Nielsen & Smith, 2014) (keduanya berbahasa Inggris).
